## 주요 기능 및 및 특징
### 1. AI 모델을 활용한 사진 설명 생성
사진 촬영시 AI 모델이 사진에 대한 설명을 생성합니다. 
### 2. 사용자 친화적인 UI/UX
시각 장애인분들을 배려한 UI/UX를 구성하였습니다.  

탭, 더블 탭 등 간단한 동작만으로 모든 기능을 실행할 수 있습니다.  

화면 전환 및 버튼 클릭 시 현재 진행 상황을 음성으로 확인 할 수 있습니다.

### 3. 저장 및 관리 기능
언제든 저장된 이미지에 대한 설명을 음성으로 확인할 수 있습니다.
